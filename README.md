This repository contains all coursework, assignments, and projects completed as part of MGT 6203: Data Analytics in Business, a core course in the Master of Science in Analytics program at Georgia Tech.

The course focuses on applying data analytics to business decision-making. Topics include data collection and preparation, exploratory data analysis, statistical analysis, regression modeling, forecasting, optimization, and data-driven strategy, with an emphasis on translating analytical insights into actionable business recommendations.
